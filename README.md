## git command
```
git add . //添加修改文件
git status //查看文件状态
git commit -m "log msg" //提交到本地仓库
git push //提交到远程master

git pull origin gh-pages //拉取到gh-pages分支
git checkout gh-pages //切换到gh-pages分支
git branch gh-pages //创建gh-pages分支
```
